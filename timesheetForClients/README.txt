Для работы системы необходимо создать каталог, в который должны быть помещены все файлы содержащие *employee*.xlsm
Шаблон для создания файлов находится в каталоге template.
dictionaries.xlsx для задач и проектов
timesheet_employee.xlsm для недель и добавления задач
После сбора всех данных и запуска программы необходимо сначала выбрать каталог с помощью соответсвующей кнопки и следом выбрать дату из календаря (можно наоборот)
После нажать кнопку "Сформировать отчёты".
Через несколько секунд в выбранном каталоге создадутся файлы по шаблону timesheet_customer_{имя проекта}.xlsx
